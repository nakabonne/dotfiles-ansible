# dotfiles-ansible
My dotfiles powered by Ansible.

## Usage

```
$ git clone https://github.com/nakabonne/dotfiles-ansible.git ~/.dotfiles
$ cd ~/.dotfiles/init
$ chmod +x setup.sh
$ ./setup.sh
```

## Supported Destribution

- Ubuntu 18.04
- macOS
